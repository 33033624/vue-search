# vue-demo之搜索大全

### 用到知识:
* vuejs2.0
* vue-resource
* 组件模块化

适合刚上手vue的童鞋。

### 如何运行
将本项目下载的本地.
> 1. 安装git
> 2. 安装nodejs
> 3. 配置node环境变量(百度一下)
> 4. 命令行进入本案例目录
> 5. 输入命令npm run install(最好使用cnpm淘宝镜像)
> 6. 输入命令npm run dev
